How it works
Three steps, zero effort
01

Connect your repo

Sign in with GitHub. RelaxRelease installs a webhook on any repository you choose — one click, nothing to configure.

02

Push a version tag

Tag your release as you normally would. RelaxRelease listens for the event and picks up automatically.

03

Review the draft

An AI-generated release is waiting in your GitHub Releases as a draft. Edit or publish — your call.

Features
Built for developers who ship
⚡

Instant on tag push

No manual trigger. Webhook fires on every tag push and your release notes are ready before you switch tabs.

🧠

AI that understands code

Powered by Google Gemini. Reads commit messages, groups changes, and writes release notes that actually make sense.

📋

Minor and major releases

Minor releases get clean changelogs. Major releases get migration guides, upgrade docs, and marketing notes.

📬

Email notifications

Get notified when a release is processed. Your team stays in the loop without checking GitHub manually.

🔒

Secure webhook verification

Every webhook is verified with HMAC-SHA256 signature checking. Only GitHub can trigger your releases.

🗂️

Full release history

Every release is stored in your dashboard. Search, review, and link to any past release notes in one place.
